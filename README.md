# Online Bookstore SQL Project

## Project Overview

This project analyzes an online bookstore database using SQL. The database contains information about books, customers, and orders. Various SQL queries were used to generate business insights and perform sales analysis.

## Database Tables

### Books

* Book_ID
* Title
* Author
* Genre
* Published_Year
* Price
* Stock

### Customers

* Customer_ID
* Name
* Email
* Phone
* City
* Country

### Orders

* Order_ID
* Customer_ID
* Book_ID
* Order_Date
* Quantity
* Total_Amount

## SQL Concepts Used

* CREATE TABLE
* PRIMARY KEY
* FOREIGN KEY
* JOINS
* GROUP BY
* HAVING
* ORDER BY
* Aggregate Functions (SUM, AVG, COUNT)
* DISTINCT
* Filtering using WHERE

## Business Problems Solved

1. Find books by genre.
2. Identify books published after a specific year.
3. Analyze customer locations.
4. Calculate total revenue.
5. Find best-selling books.
6. Identify top customers by spending.
7. Analyze genre-wise sales.
8. Calculate remaining stock after orders.

## Dataset Files

* Books.csv
* Customers.csv
* Orders.csv

## Project Outcome

This project demonstrates SQL skills for data analysis, database design, data filtering, aggregation, and business reporting.
